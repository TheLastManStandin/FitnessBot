# Информация для Claude

## Технологии проекта

### Версии библиотек
- **Aiogram**: версия 3 (aiogram3), НЕ версия 2
- **FastAPI**: последняя версия
- **Python**: 3.10+

### Особенности aiogram3
- Используется современный синтаксис aiogram 3.x
- Все обработчики должны быть совместимы с aiogram3
- Проверять актуальность кода при добавлении новых функций

### Структура проекта
- Микросервисная архитектура
- RabbitMQ для межсервисной коммуникации
- Docker контейнеризация

## Рекомендации по разработке
- Проверять совместимость кода с aiogram3
- Использовать async/await для всех операций
- Следовать модульной архитектуре проекта

## Рекомендации для тестирования
- При запуске python используй python3
